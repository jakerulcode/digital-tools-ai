## 📖 About The Project

**Digital Tools AI** is a sleek digital product store built with React. Users can browse a curated collection of premium AI tools, add them to cart, and enjoy a smooth shopping experience — all powered by local JSON data and a modern UI stack.

> Built to demonstrate React fundamentals, state management, and responsive UI design.

---

## ✨ Features

| Feature | Description |
|--------|-------------|
| 🤖 AI Tools Collection | Browse and discover a curated list of premium AI tools |
| 🛒 Cart System | Add, remove, and manage products with live cart count |
| 🔒 End-to-End Encryption | Fully protected with industry-grade encryption |
| ☁️ 1 TB Cloud Storage | Ample space to store all your digital assets securely |
| 📱 5 Device Sync | Stay in sync across up to 5 devices in real time |
| 🔔 Toast Notifications | Real-time feedback on every user action |
| 📱 Responsive Design | Fully optimized for mobile, tablet, and desktop |

## 🛠️ Installation & Setup
```bash
# Clone the repository
git clone https://github.com/jakerulcode/digital-tools-ai.git

# Navigate into the project
cd digital-tools-ai

# Install dependencies
npm install

# Start the development server
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173) in your browser.
